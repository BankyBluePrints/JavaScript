# ES6 / ES2015 (ECMAScript 2015)

**Release Year:** 2015

## Major Highlights

- **let and const** - Block-scoped variables
- **Arrow Functions** - Concise function syntax
- **Classes** - Syntactic sugar for OOP
- **Template Literals** - String interpolation
- **Destructuring** - Extract values from arrays/objects
- **Default Parameters** - Function parameter defaults
- **Rest/Spread Operators** - Flexible function arguments
- **Promises** - Better async programming
- **Modules** - Import/export system
- **Iterators & Generators** - Custom iteration behavior
- **Map, Set, WeakMap, WeakSet** - New data structures
- **Symbols** - Unique property keys
- **for...of Loop** - Iterate over iterables

## Why This Version Changed How JavaScript Is Written

ES6/ES2015 was the **biggest update** to JavaScript since ES5 (2009):

- **Block scoping** eliminated confusing `var` behavior
- **Arrow functions** solved the `this` binding problem
- **Classes** brought familiar OOP syntax
- **Modules** enabled proper code organization
- **Promises** made async code manageable
- **Template literals** replaced string concatenation
- **Destructuring** simplified data extraction

This version modernized JavaScript to match other contemporary programming languages while maintaining backward compatibility. ES6 marked the beginning of yearly ECMAScript releases and established JavaScript as a mature, full-featured programming language.
