# ES2021 (ECMAScript 2021)

**Release Year:** 2021

## Major Highlights

- **String.replaceAll()** - Replace all occurrences
- **Promise.any()** - First fulfilled promise
- **Logical Assignment Operators** - `&&=`, `||=`, `??=`
- **Numeric Separators** - `1_000_000` for readability
- **WeakRefs** - Weak references to objects
- **FinalizationRegistry** - Cleanup callbacks for garbage collection

## Why This Version Changed How JavaScript Is Written

ES2021 added convenient quality-of-life improvements:

- **replaceAll()** fixed a longstanding annoyance
- **Logical assignment** shortened common patterns
- **Numeric separators** improved large number readability
- **Promise.any()** completed the Promise utilities

While not revolutionary, these features made daily coding more pleasant and reduced common mistakes.
