# ES2019 (ECMAScript 2019)

**Release Year:** 2019

## Major Highlights

- **Array.flat()** - Flatten nested arrays
- **Array.flatMap()** - Map and flatten in one operation
- **Object.fromEntries()** - Convert entries to object
- **String.trimStart() / trimEnd()** - Trim from specific side
- **Optional catch binding** - Catch without parameter
- **Symbol.description** - Access symbol description
- **Well-formed JSON.stringify()** - Proper Unicode handling

## Why This Version Changed How JavaScript Is Written

ES2019 added practical array and object manipulation tools:

- **Array.flat()** simplified working with nested data structures
- **Object.fromEntries()** complemented Object.entries() perfectly
- **trimStart/trimEnd** provided fine-grained string control
- **Optional catch** reduced boilerplate in error handling

These features made common operations more elegant and less verbose.
