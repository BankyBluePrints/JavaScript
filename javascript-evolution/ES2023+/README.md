# ES2023+ (ECMAScript 2023 and Beyond)

**Release Year:** 2023 and ongoing

## Major Highlights (ES2023)

- **Array.findLast() / findLastIndex()** - Find from end
- **Array.toReversed()** - Immutable reverse
- **Array.toSorted()** - Immutable sort
- **Array.toSpliced()** - Immutable splice
- **Array.with()** - Immutable element replacement
- **Hashbang Grammar** - `#!/usr/bin/env node` support
- **Symbols as WeakMap keys** - Use symbols in WeakMaps

## Upcoming Features (Proposals)

- **Temporal API** - Modern date/time handling
- **Decorators** - Class and method decorators
- **Pattern Matching** - Match expressions
- **Records and Tuples** - Immutable data structures
- **Pipeline Operator** - `value |> fn`

## Why These Features Matter

ES2023 focused on **immutable array methods**, addressing a gap in array manipulation:

- Previous methods mutated arrays
- New methods return new arrays (safer, functional)
- **findLast()** addressed common search pattern

**Future features** will bring even more power:
- Temporal fixes Date's long-standing issues
- Decorators simplify meta-programming
- Pattern matching makes complex conditionals cleaner

JavaScript continues to evolve annually with focused, practical improvements!
